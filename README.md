# Jithu Kannan Portfolio - EEE & AI

A modern, cozy, and refreshing personal portfolio website designed for an Electrical & Electronics Engineering student specializing in AI, Embedded Systems, and IoT.

## 🚀 Features
- **Cozy Design**: Warm pastel color palette (#F6F1EB, #A8C3A0, #7FA7B8).
- **Responsive**: Mobile-first design with a clean, grid-based layout.
- **Interactive**: Smooth scroll animations, glassmorphism effects, and hover transitions.
- **Tech Stack**: HTML5, CSS3, JavaScript (Vanilla).

## 🛠️ Local Setup
1. Clone the repository.
2. Open `index.html` in your browser.
3. (Optional) A Python virtual environment has been created in `.venv` for any AI/ML scripts or calculations you wish to add.

## 📦 Deployment
This portfolio is "ready to deploy". You can use any static hosting service:
- **GitHub Pages**: Upload these files to a repo and enable Pages in settings.
- **Vercel**: Run `vercel` in this directory or connect your GitHub repo.
- **Netlify**: Drag and drop the folder into the Netlify dashboard.

## 📂 Project Structure
- `index.html`: Main structure.
- `style.css`: All styling and animations.
- `main.js`: Interactivity and scroll logic.
- `.venv/`: Python virtual environment for your AI/EEE projects.
- `requirements.txt`: Placeholder for Python dependencies.
