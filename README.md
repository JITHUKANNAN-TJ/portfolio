# Jithu Kannan Portfolio - EEE & AI

A premium, cinematic personal portfolio website designed for an Electrical & Electronics Engineering student specializing in AI, Embedded Systems, and IoT.

## 🚀 Features
- **Cinematic Theme**: Attractive high-contrast Black & Red design.
- **Responsive**: Mobile-first design with a clean, grid-based layout.
- **Interactive**: Smooth scroll animations, refined project hover effects, and a professional resume download.
- **Tech Stack**: HTML5, CSS3, JavaScript (Vanilla).

## 🛠️ Local Setup
1. Clone the repository: `git clone https://github.com/JITHUKANNAN-TJ/portfolio.git`
2. Open `index.html` in your browser.
3. (Optional) Run Python scripts in the `.venv` for AI/ML tasks.

## 📦 How to Deploy (GitHub Pages)
Your code is already pushed to GitHub! To make the website live:

1. Go to your repository on GitHub: [portfolio](https://github.com/JITHUKANNAN-TJ/portfolio)
2. Click the **Settings** tab.
3. Click **Pages** in the left sidebar.
4. Under **Build and deployment > Branch**, select **main** (or `master`) and click **Save**.
5. Wait ~1 minute, and your site will be live at:  
   👉 `https://JITHUKANNAN-TJ.github.io/portfolio/`

## 📂 Project Structure
- `index.html`: Main structure and hero section.
- `style.css`: All styling, Black & Red theme, and animations.
- `main.js`: Scroll logic and interactivity.
- `profile.png`: Your professional profile photo.
- `hero-img.png`: The custom cinematic hero illustration.
- `resume.pdf`: Your integrated professional resume.
